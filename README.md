# pi-zero-to-robot 🎤🤖

> 喊一声 **"土豆"**，树莓派就能放歌给你听。

一个运行在树莓派上的语音助手，从零开始搭建。不需要编程经验，不需要屏幕，**30 分钟从拆快递到喊出第一句**。

> **💰 费用说明：** 本项目开源免费，但调用外部 API 会产生费用。
> - **百度语音**（STT/TTS）：免费，有免费额度
> - **DeepSeek 聊天**：付费，按量计费，充值 1 元起，聊天几百次仅几分钱
> - **网易云音乐**：免费歌曲免费播放，VIP 歌曲需登录+会员
>
> 除此之外无任何隐藏费用，硬件成本见第一步。
