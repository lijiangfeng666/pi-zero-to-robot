# pi-zero-to-robot 🎤🤖

> **喊一声"土豆"，树莓派就能放歌给你听。**

一个运行在树莓派上的**AI 语音机器人**，从零开始搭建。不需要编程经验，不需要给树莓派接屏幕，**跟着步骤走就行**。

---

## 🧐 这个机器人能做什么？

| 你说 | 它做什么 |
|------|---------|
| 🎵 **"土豆放一首大海"** | 搜索并播放《大海》 |
| ⏭ **"土豆下一首"** | 切歌 |
| ⏸ **"土豆暂停"** | 暂停播放 |
| ▶️ **"土豆继续"** | 恢复播放 |
| 🔊 **"土豆大声点"** | 调高音量 |
| 🌤 **"土豆今天天气"** | 播报天气 |
| 💬 **"土豆今天心情怎么样"** | 跟你聊天 |

> 全程**语音控制**，不需要手机 App，不需要按键。

- ✅ 一张 **SD 卡** + 一个 **读卡器**（或电脑自带卡槽）

## 📁 项目文件说明

| 文件 | 它管什么 |
|------|---------|
| `main.py` | 机器人的大脑——监听声音、调度一切 |
| `voice.py` | 耳朵和嘴巴——录音、语音识别、语音合成 |
| `intent_parser.py` | 理解你——把你说的话翻译成指令 |
| `executor.py` | 执行指令——放歌、调音量、查天气等 |
| `music_player.py` | 音乐播放器——控制播放、暂停、切歌 |
| `netease.py` | 音乐搜索——从网易云音乐找歌 |
| `config.py` | 配置读取——加载你的 API Key |
| `.env.example` | 配置模板——复制成 `.env` 填入密钥 |

## 🔰 开始之前

如果你是**第一次接触树莓派**，有几个事情先了解一下：

**❓ 我不需要懂编程吧？**
不需要。你只需要**复制粘贴命令**，每一步都写好了。

**❓ 会折腾坏树莓派吗？**
不会。系统装坏了重新烧 SD 卡就行，树莓派本身不会坏。

**❓ 我连终端是什么都不知道……**
没关系。第3步会教你打开什么软件、点什么按钮、输入什么命令。每一步都写明了**在哪个窗口**操作。

**❓ 需要联网吗？**
需要。树莓派要连 WiFi 才能用语音识别和 AI 对话。
