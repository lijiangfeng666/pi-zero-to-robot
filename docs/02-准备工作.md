# 第二步：准备工作

> 所有在电脑上要提前准备好的东西。**在你拿到树莓派之前就可以做。**

全程在**你自己的电脑**（Windows/Mac）上操作，不需要树莓派。

---

## 📋 准备工作清单

| # | 做什么 | 需要多久 | 在哪做 |
|:--:|-------|---------|--------|
| 2.1 | 检查电脑是否满足要求 | 1 分钟 | 你的电脑 |
| 2.2 | 确认家里有 WiFi | 1 分钟 | 你家 |
| 2.3 | 注册百度智能云账号 | 5 分钟 | 浏览器 |
| 2.4 | 注册 DeepSeek 账号 | 5 分钟 | 浏览器 |

## 2.1 检查电脑是否满足要求

**Windows 用户：**
- ✅ 系统：Windows 10 或 Windows 11
- ✅ 有一个空闲的 USB 口（插读卡器用）
- ✅ 能连上家里的 WiFi

**Mac 用户：**
- ✅ 系统：macOS 10.15（Catalina）或更新版本
- ✅ 有一个空闲的 USB 口
- ✅ 能连上家里的 WiFi

> 如果你不确定自己的 Windows 版本：按住 `Win + R` → 输入 `winver` → 回车，会弹出窗口告诉你版本号。

---

## 2.2 确认家里有 WiFi

树莓派需要**通过 WiFi 上网**才能用语音识别和 AI 对话。

```
☐ 你家的 WiFi 名字（SSID）是什么？    写在这里：___________
☐ 你家的 WiFi 密码是什么？             写在这里：___________
```

> 后面烧录系统的时候要填这两个信息，提前找出来省得翻箱倒柜找路由器。

**最好现在就记下来：**

> WiFi 名（SSID）：_______________
> WiFi 密码：___________________

> ⚠️ WiFi 名**区分大小写**（"TP-LINK_5G" 和 "tplink_5g" 不一样），请原样抄下来。
> ⚠️ WiFi 密码不要有空格（如果密码里本来就有空格，那也要原样输入）。

---

## 2.3 注册百度智能云账号

> 用来让机器人的"耳朵"（语音识别）和"嘴巴"（语音合成）工作。

**① 打开浏览器**（Chrome、Edge、Safari 都可以）

- **Windows 电脑：** 双击桌面上的 Chrome 蓝色圆形图标，或 Edge 蓝色/绿色图标
- **Mac 电脑：** 点屏幕下方的 Safari 图标（蓝色罗盘）

**② 访问百度智能云控制台**

在浏览器顶部的地址栏输入以下地址，然后按键盘上的 **回车键**：

```
https://console.bce.baidu.com/
```

**③ 注册/登录**

```
如果在登录页面：
  → 点「立即注册」（如果还没有百度账号）
  → 用手机号注册（跟注册微信号一样）
  → 注册完登录

如果已经有百度账号（比如百度网盘）：
  → 直接登录
```

**④ 进入语音技术控制台**

```
登录后 → 在搜索框输入「语音技术」
  → 点「语音技术」进入
  → 看到「创建应用」按钮就对了
```

> 注册好就行，**不需要充值**。百度语音识别免费的额度足够个人用。

---

## 2.4 注册 DeepSeek 账号

> 用来让机器人能理解你的指令和聊天。

**① 访问 DeepSeek 平台**

在浏览器顶部地址栏输入：

```
https://platform.deepseek.com/
```

**② 注册账号**

```
  → 点「Sign Up」或「注册」
  → 用邮箱注册（QQ邮箱、163邮箱都可以）
  → 填写验证码
  → 登录
```

> 注册好就行，**不需要充值**。DeepSeek 新用户有免费额度。
> 如果看不懂英文，可以点页面右上角切换成中文。

---

## 2.5 下载烧录工具（Raspberry Pi Imager）

> 后面第3步要用这个工具往 SD 卡里装系统。**提前下好，省得第3步干等。**

**① 访问下载页面**

在浏览器地址栏输入：

```
https://www.raspberrypi.com/software/
```

> 如果打不开，试试把网址复制粘贴到地址栏，不要手动输入。

**② 下载对应版本**

| 你的电脑 | 点哪个按钮 |
|---------|-----------|
| Windows | 蓝色的 **Download for Windows** |
| Mac（Intel 芯片） | 蓝色的 **Download for macOS** |
| Mac（Apple 芯片 M1/M2/M3） | 下面还有一行小字：**macOS arm64** |

> 不确定 Mac 芯片类型：点屏幕左上角苹果图标 → 「关于本机」→ 看芯片显示的是 Intel 还是 Apple M1/M2/M3

**③ 保存文件**

下载的文件通常出现在浏览器左下角（Windows）或下载文件夹（Mac）。

**④ 查看下载位置（确认一下）**

- **Windows：** 文件名叫 `raspberry-pi-imager.exe`
- **Mac：** 文件名叫 `Raspberry-Pi-Imager.dmg`

```
☐ 确定已经下载到电脑上了
```

> **不用急着安装**，第3步会一步步教你怎么装。
