# 第二步：准备工作

除了树莓派的硬件，你还需要一台**电脑**（Windows 或 Mac），用来给 SD 卡装系统和远程连接树莓派。

全程不需要给树莓派接显示器——所有操作都在电脑上完成。

**✅ 确认有一台能上网的电脑（Windows/Mac）→ 继续下一步。**

---

## 2.1 找到你家 WiFi 的名字和密码

> 树莓派需要联网，烧录系统时要填 WiFi 信息。

**Windows 电脑查看 WiFi 密码：**

```
① 点电脑屏幕右下角的网络图标（像一个小地球🌐或扇形📶）
② 点「网络和 Internet 设置」
③ 点「高级网络设置」→「更多网络适配器选项」
   弹出"网络连接"窗口
④ 双击你家 WiFi 的名字（蓝色字）
⑤ 点「无线属性」
⑥ 点上面的「安全」选项卡
⑦ 勾选「显示字符」
   密码那一栏就显示出来了 —— 记下来
```

**把 WiFi 信息写在这里（后面烧录系统要用）：**

```
WiFi 名称（SSID）：_______________
WiFi 密码：_______________
```

> ⚠️ WiFi 名称**区分大小写**，`TP-LINK` 和 `tp-link` 不一样，请原样抄写。

---

## 2.2 注册百度智能云账号

> 让机器人能听懂人话（语音识别）和开口说话（语音合成）。

```
① 在电脑上找到浏览器图标（Chrome 蓝色圆形 / Edge 蓝色/绿色图标）
② 双击鼠标左键打开浏览器
③ 在顶部地址栏输入以下地址，按键盘回车键：
   https://console.bce.baidu.com/
④ 点右上角的「立即注册」（如果已有百度账号，点「登录」）
⑤ 用手机号注册 → 收验证码 → 设置密码
⑥ 注册成功后自动登录
```

> 不需要充值，免费额度够用。

---

## 2.3 注册 DeepSeek 账号

> 让机器人能聊天和理解指令。

```
① 在浏览器顶部地址栏输入：
   https://platform.deepseek.com/
② 按键盘回车键
③ 点「Sign Up」或「注册」
④ 输入邮箱地址（QQ邮箱、163邮箱都可以）
⑤ 收验证码 → 设置密码
⑥ 注册成功后登录
   （页面右上角会显示你的头像或邮箱）
```

> 不需要充值，新用户有免费额度。

---

## 2.4 下载烧录工具

> 提前下载好，第3步要用。

```
① 在浏览器顶部地址栏输入：
   https://www.raspberrypi.com/software/
② 按键盘回车键
   页面加载出来后，看到蓝色的按钮
③ Windows 用户点「Download for Windows」
   Mac 用户点「Download for macOS」
   文件开始下载 —— 浏览器左下角（或右上角）会显示进度
④ 等下载完成（文件大约 30MB，约 1-2 分钟）
   Windows 用户看到文件：raspberry-pi-imager.exe
   Mac 用户看到文件：Raspberry-Pi-Imager.dmg
```

> 不用安装，第3步会教你怎么装。
