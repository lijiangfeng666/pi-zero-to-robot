# 第七步：测试语音

> 测试麦克风和音箱是否正常工作。
>
> 📍 **在哪操作：🟢 树莓派窗口**（提示符 `pi@pi-zero:~ $`）
>
> 这一步全部在树莓派窗口完成。

---

## 7.1 确认麦克风已连接

> 看看树莓派有没有识别到 USB 麦克风。

在树莓派窗口输入：

```bash
arecord -l
```

按回车。

**✅ 如果看到类似这样的输出：**

```
card 3: WebCamera [USB Camera]...
```

说明麦克风被识别了（`card 3` 前面的数字可能是 2、3、4，没关系）。

**❌ 如果看到：**

```
no soundcards found...
```

说明没识别到麦克风。试一下：

```
① 把麦克风拔下来，换一个 USB 口插回去
② 再执行 arecord -l 看看有没有变化
③ 如果还是不行，重启树莓派：sudo reboot
```

---

## 7.2 确认音箱已连接

> 看看树莓派有没有识别到 USB 音箱。

在树莓派窗口输入：

```bash
aplay -l
```

**✅ 看到类似：**

```
card 4: USB2.0 Device [USB Audio]...
```

说明音箱被识别了。

---

## 7.3 （了解一下）声卡映射

执行 `arecord -l` 和 `aplay -l` 后，你可能会看到多个声卡：
```
card 0: vc4hdmi0      — HDMI 输出（用不到）
card 1: vc4hdmi1      — HDMI 输出（用不到）
card 2: bcm2835       — 树莓派自带的 3.5mm 耳机口
card 3: WebCamera     — USB 麦克风（语音输入用的）
card 4: USB2.0 Device — USB 音箱（语音输出用的）
```

> 不同品牌的 USB 设备编号可能不同（比如你的麦克风可能是 card 2），这不影响使用，机器人会自动识别。

---

## 7.4 测试录音和回放

> 录一段声音，然后放出来听。

**第一步：录音 3 秒**

```bash
arecord -d 3 -f S16_LE -r 16000 /tmp/test.wav
```

对着麦克风说话（随便说点啥），3 秒后自动结束。

**第二步：回放刚才录的声音**

```bash
aplay /tmp/test.wav
```

**✅ 能听到自己刚才说的话 → 麦克风和音箱都正常。**

**❌ 回放没声音：**

```
① 检查音箱：通电了没？音量开关开了没？
② 调音量：在树莓派窗口输入 alsamixer → 用 ↑ 方向键把音量调到 70% → 按 Esc 退出
③ 如果还是没声，试试重启树莓派：sudo reboot
```

---

## 7.5 测试语音识别（STT）

> 你说话 → 百度语音识别成文字。这是机器人"听懂"你的关键一步。

在树莓派窗口输入（复制整段，粘贴进去）：

```bash
cd ~/robot && python3 -c "
from voice import VoiceEngine
v = VoiceEngine()
print('录音中，说句话...')
audio = v.record_audio(3)
text = v.speech_to_text(audio)
print('你说的是：' + (text or '没识别到'))
v.pa.terminate()
"
```

按回车。看到 `录音中，说句话...` 后，对着麦克风说 **"你好"**。

**✅ 看到类似这样：**

```
录音中，说句话...
你说的是：你好
```

→ 语音识别正常！

**❌ 看到 `你说的是：没识别到`：**

```
① 靠近麦克风大声点再试一次
② 检查网络：ping baidu.com 看看通不通
③ 确认 .env 里的百度 AppID、API Key、Secret Key 都填对了
```

---

## 7.6 测试语音合成（TTS）

> 让树莓派"开口说话"，把文字变成语音播出来。

在树莓派窗口输入：

```bash
cd ~/robot && python3 -c "
from voice import VoiceEngine
v = VoiceEngine()
v.text_to_speech('你好，我是你的语音助手')
v.pa.terminate()
"
```

**✅ 音箱传出声音「你好，我是你的语音助手」→ TTS 正常。**

**❌ 没声音：** 检查音箱是否通电、音量是否太小。

---

## 7.7 一键验证百度语音 Key

> 快速检查 Key 是否有效。

```bash
cd ~/robot && python3 -c "
from aip import AipSpeech
from dotenv import load_dotenv
import os
load_dotenv()
c = AipSpeech(os.getenv('BAIDU_APP_ID'), os.getenv('BAIDU_API_KEY'), os.getenv('BAIDU_SECRET_KEY'))
r = c.synthesis('测试', 'zh', 1, {'vol': 5})
print('百度语音 Key', '✅' if not isinstance(r, dict) else '❌ 无效')
"
```

**✅ 看到 `百度语音 Key ✅` → 全部正常。**
