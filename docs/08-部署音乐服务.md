# 第八步：部署音乐服务

> 部署网易云音乐的 API 服务器，让机器人能搜歌和放歌。
>
> 📍 **在哪操作：🟢 树莓派窗口**（提示符 `pi@pi-zero:~ $`）

---

## 8.1 安装网易云 API 服务器

> 在树莓派上下载并安装一个"音乐翻译官"，负责把机器人的"搜歌"指令变成网易云音乐能懂的请求。

```
① 在树莓派窗口复制粘贴下面命令：
   cd ~/robot && git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git netease-server && cd netease-server && npm install
② 按键盘回车键
   看到一堆文字在滚动 —— 正在下载和安装
   （这条命令做了三件事：下载代码 → 进入文件夹 → 安装依赖）
```

⏳ **这一步比较慢，树莓派 4B 约 2-3 分钟，老款（Zero/3B）会更久。** 跑完才能进入下一步，耐心等它完成就好。

看到类似这样的输出就是装完了：

```
added 1000+ packages in 30s
```

就安装完了。

**如果安装很慢（国内网络原因）：**

先按 `Ctrl+C` 取消当前命令，然后换国内镜像再试：

```bash
cd ~/robot && rm -rf netease-server && git clone https://github.com/Binaryify/NeteaseCloudMusicApi.git netease-server && cd netease-server && npm config set registry https://registry.npmmirror.com && npm install
```

## 8.2 临时启动并测试

> 先手动启动音乐服务器，看看能不能正常工作。

```
① 在树莓派窗口复制粘贴下面命令：
   cd ~/robot/netease-server && node app.js
② 按键盘回车键
   看到 Server running @ http://localhost:3000
   —— 音乐服务器启动成功
   （注意：这个窗口被占住了，不要再输入命令，保持不动）
```

**第二步：再开一个树莓派窗口来测试**

> 因为刚才的终端被音乐服务器占住了，你需要**新开一个连接**来测试。

```
① 在电脑上找到之前的 💻 电脑窗口（或新开一个：Win + R → powershell）
② 复制粘贴下面命令（把 IP 换成你的）：
   ssh pi@192.168.1.100
③ 按键盘回车键
④ 输入密码 raspberry，按回车
   看到 pi@pi-zero:~ $ —— 这是新树莓派窗口
```

**第三步：在新树莓派窗口测试服务器**

```
① 在新树莓派窗口复制粘贴下面命令：
   curl http://127.0.0.1:3000
② 按键盘回车键
   看到一堆以 { 开头的内容（JSON 数据）
   —— 音乐服务器跑起来了
```

## 8.3 测试音乐搜索

```
① 在新树莓派窗口（就是刚才 SSH 连的那个）复制粘贴下面命令：
   cd ~/robot && python3 -c "
   from netease import NetEaseClient
   n = NetEaseClient()
   songs = n.search('大海')
   if songs:
       print(f'搜索 ✅: {songs[0][\"name\"]} - {songs[0].get(\"artist\", \"\")}')
   else:
       print('搜索失败，API 服务没启动？')
   "
② 按键盘回车键
   看到：搜索 ✅: 大海 - 张雨生
   —— 音乐搜索正常

> **关于免费和VIP：** 网易云大部分歌曲免费播放，不需要登录。如果遇到"播不出声"或"只播几秒就停"，通常是那首歌需要 VIP 会员。登录网易云账号后（见 8.6），VIP 歌曲也能正常播放。
```

## 8.4 测试音乐播放

确保音箱是通电状态，音量调到适中。

```
① 在新树莓派窗口复制粘贴下面命令：
   cd ~/robot && python3 -c "
   from netease import NetEaseClient
   from music_player import MusicPlayer
   n = NetEaseClient()
   m = MusicPlayer(netease_client=n)
   print(m.play('大海'))
   import time
   time.sleep(10)
   m.stop()
   "
② 按键盘回车键
   音箱响起音乐（10 秒后自动停止）
   —— 音乐模块完整可用
```

**测试完后**，回到第一个被占住的树莓派窗口，按 `Ctrl+C` 关掉音乐服务器。

## 8.5 音乐服务开机自启

> 配置好后，树莓派每次开机都会自动启动音乐服务器，不需要手动开。

```
① 在树莓派窗口复制粘贴下面整段命令（从 mkdir 到 EOF 全部选中）：
   mkdir -p ~/.config/systemd/user/ && cat > ~/.config/systemd/user/netease-api.service << 'EOF'
   [Unit]
   Description=NetEase Cloud Music API
   After=network.target
   
   [Service]
   Type=simple
   WorkingDirectory=%h/robot/netease-server
   ExecStart=/usr/bin/node app.js
   Restart=on-failure
   RestartSec=5
   
   [Install]
   WantedBy=default.target
   EOF
② 按键盘回车键
   没有输出 —— 说明服务文件创建成功
③ 复制粘贴下面命令，按回车：
   systemctl --user daemon-reload
   没有输出 —— 重新加载服务配置
④ 复制粘贴下面命令，按回车：
   systemctl --user enable --now netease-api.service
   看到提示 Created symlink... 或类似输出 —— 开机自启已启用
⑤ 复制粘贴下面命令，按回车：
   loginctl enable-linger
   没有输出 —— 允许后台运行
```

**验证服务是否启动成功：**

```
⑥ 复制粘贴下面命令，按回车：
   systemctl --user status netease-api.service
```

**✅ 看到 `active (running)`（绿色字）→ 音乐服务开机自启配置完成！**

> 如果看到红色的 `inactive (dead)` 或 `failed`，检查一下上一步的 rm -rf 是不是把文件夹删了：
> ```bash
> ls ~/robot/netease-server
> ```
> 看有没有文件，如果空了就重新执行 8.1。

## 8.6 登录网易云账号

> 让机器人能搜索和播放网易云音乐，需要先登录你的账号。
>
> 📍 **同时用到两个窗口：🟢 树莓派窗口 + 💻 电脑窗口**

**第一步：确认音乐服务正在运行**

在 🟢 树莓派窗口输入：
```bash
systemctl --user status netease-api.service
```
**确认看到绿色的 `active (running)`**，然后继续。

**第二步：在电脑上打开登录页面**

在 **💻 电脑窗口**输入（**将 `192.168.1.100` 换成你的树莓派 IP**）：
```bash
ssh -L 3000:localhost:3000 pi@192.168.1.100
```
> 这条命令的意思：把电脑的 3000 端口和树莓派的 3000 端口打通，这样你电脑的浏览器就能访问树莓派上的音乐 API 服务了。

输入密码 `raspberry` 后，这个窗口会保持连接，**不要关**。

**第三步：打开浏览器登录**

在电脑上打开浏览器，地址栏输入：
```
http://localhost:3000
```
> 看到页面后，往下翻找到「登录」区域。点显示二维码的按钮，打开手机上的网易云音乐 App，扫二维码登录。

**第四步：验证登录成功**

回到 **💻 电脑窗口**（刚才 SSH 的那个），先按 `Ctrl+C` 退出 SSH。
然后在电脑窗口输入：
```bash
ssh pi@192.168.1.100 "cd ~/robot && python3 -c 'from netease import NetEaseClient; n = NetEaseClient(); print(\"✅ 已登录\" if n.check_login() else \"❌ 未登录\")'"
```

**✅ 看到 `✅ 已登录` → 登录成功！**
机器人会每 12 小时自动刷新一次 Cookie（有效期 180 天），以后不用重复登录。

> ⚠️ 如果没有登录直接搜歌，机器人会说"好嘞"但音箱不响（因为未登录拿不到播放链接）。
