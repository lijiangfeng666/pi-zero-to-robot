# 第六步：部署代码和配置密钥

> 把机器人代码下载到你的电脑上，然后传到树莓派里。
>
> 这一步需要同时用到**两个窗口**：
> 1. **电脑窗口**（你电脑自己的终端）——下载代码和传文件
> 2. **树莓派窗口**（`pi@pi-zero:~ $`）——接收文件、配置密钥
>
> **不要搞混了！** 两个窗口看起来很像，注意看提示符。

---

## 🔸 先把两个窗口都准备好

### 窗口一：树莓派窗口（已经有的，别关）

如果你一直在跟着做，从第5步到现在，你的终端应该是树莓派窗口（显示 `pi@pi-zero:~ $`）。

**这个窗口保持不动，别关。**

### 窗口二：电脑窗口（需要新开一个）

> 现在你需要**再开一个**终端窗口——这是你**自己电脑**的终端，不是树莓派的。

#### Windows 用户——新开一个电脑终端

```
① 按住 Win 键 + R 键
② 输入：powershell
③ 点「确定」
   弹出深蓝色背景的窗口 —— 这是你的电脑窗口
   （看提示符，如果是 C:\Users\你的名字> 就对了）
```

#### Mac 用户——新开一个电脑终端

```
① 点屏幕右上角放大镜
② 搜索"终端"
③ 点「终端」打开
   弹出白/黑色背景的窗口 —— 这是你的电脑窗口
   （看提示符，如果是 你的名字@你的电脑名 ~ % 就对了）
```

> 现在你桌面上应该有两个终端窗口：
> - **一个深色背景的**（树莓派窗口，显示 `pi@pi-zero:~ $`）
> - **一个刚打开的**（电脑窗口，显示你电脑的用户名）
>
> 后面我会用 💻「电脑窗口」和 🟢「树莓派窗口」来区分。

---

## 6.1 在电脑上下载代码

> 在 **💻 电脑窗口** 操作。

**第一步：决定把代码放在哪里**

建议放在**桌面**，方便找到。

```
① 确认你在 💻 电脑窗口（看提示符是 C:\Users\你的名字> 或 你的名字@电脑名）
② 复制下面命令：
   cd ~/Desktop
③ 在电脑窗口点右键粘贴（或按 Ctrl+V）
④ 按键盘回车键
   没有任何输出（直接出现下一行提示符）—— 说明已经跳到桌面了
```

**第二步：下载代码**

```
① 在电脑窗口复制粘贴下面命令：
   git clone https://github.com/lijiangfeng666/pi-zero-to-robot.git
② 按键盘回车键
   看到类似这样的输出：
   Cloning into 'pi-zero-to-robot'...
   remote: Enumerating objects: ...
   Receiving objects: ... done.
   —— 说明代码下载成功
③ 复制粘贴下面命令进入项目文件夹：
   cd pi-zero-to-robot
④ 按键盘回车键
   没有输出（直接出现下一行提示符）—— 说明已经进入项目文件夹
```

**没有装 git 怎么办？**

如果提示 `'git' 不是内部或外部命令` 或 `command not found`，用另一种方式：

```
① 打开浏览器（Chrome/Edge）
② 访问：https://github.com/lijiangfeng666/pi-zero-to-robot
③ 点绿色的「Code」按钮 → 点「Download ZIP」
④ 下载完成后，把 zip 文件解压到桌面
   右键点 zip 文件 → 选择「解压到当前文件夹」
⑤ 解压后会有一个叫 pi-zero-to-robot 的文件夹
```

## 6.2 把代码传到树莓派

> 在 **💻 电脑窗口** 操作。

```
① 复制下面这行命令（注意把里面的 192.168.1.100 换成你自己的树莓派 IP）
   scp main.py config.py voice.py music_player.py netease.py executor.py intent_parser.py .env.example pi@192.168.1.100:~/robot/
② 在电脑窗口点右键粘贴
③ 按键盘回车键
   看到：pi@192.168.1.100's password:
   —— 系统在问你树莓派的密码
④ 输入 raspberry（全小写），按回车
   注意：输入密码时屏幕什么都不会显示 —— 这是正常的
   看到每个文件名后面带 100% 的进度 —— 说明文件传成功了
```

> `scp` = Secure Copy（安全复制），通过 SSH 把文件传到树莓派。

**验证文件是否已传到树莓派：**

```
① 在电脑窗口复制粘贴下面命令（把 IP 换成你的）：
   ssh pi@192.168.1.100 "ls ~/robot/"
② 按键盘回车键
③ 输入密码 raspberry，按回车
   看到以下 8 个文件即成功：
   main.py
   config.py
   voice.py
   music_player.py
   netease.py
   executor.py
   intent_parser.py
   .env.example
```

**✅ 看到 7 个 .py 文件 + .env.example → 部署成功。**

## 6.3 申请百度语音 Key

> 申请百度语音的 API Key，让树莓派能"听懂"和"开口说话"。
> **百度语音是免费服务**，有免费额度，不用担心费用。
>
> 📍 **在电脑上打开浏览器即可，不需要终端。**

**打开浏览器，访问百度语音控制台：**

```
① 在浏览器地址栏输入：
   https://console.bce.baidu.com/ai-engine/
② 如果没登录，用手机号或百度账号登录
③ 登录后点「语音技术」
④ 点「创建应用」
⑤ 名称填：语音机器人
   勾选：语音识别 + 语音合成
⑥ 点「立即创建」
```

**创建成功后，你会看到三个关键信息——复制下来保存好（后面要用）：**

```
AppID:       12345678       ← 一串数字
API Key:     abcdefghijklmn ← 字母数字混合
Secret Key:  1234567890abcdef ← 字母数字混合
```

> ⚠️ **建议先粘贴到记事本/备忘录里保存**，免得页面关了就找不到了。

## 6.4 申请 DeepSeek Key

> 申请 DeepSeek 的 API Key，让机器人能和你聊天。
>
> ⚠️ **注意：DeepSeek API 是付费服务**，按量计费。首次使用需充值 1 元以上。
> 聊天几百次才几分钱，不用担心费用。充值入口在 API Keys 页面的右上角。

**打开浏览器，访问 DeepSeek 平台：**

```
① 在浏览器地址栏输入：
   https://platform.deepseek.com/
② 注册账号（用邮箱）→ 登录
③ 点左边菜单的「API Keys」
④ 点「创建 API Key」
⑤ 名字随便填，比如：pi-robot
```

**复制生成的 Key（以 sk- 开头的一长串字符）：**

```
sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

> ⚠️ **赶紧保存！** DeepSeek 的页面关掉后就再也看不到这个 Key 了。
> 建议复制到刚才保存百度 Key 的记事本里。

## 6.5 把 Key 填到配置文件中

> 在 **🟢 树莓派窗口**（`pi@pi-zero:~ $`）操作。
>
> 确认一下：你的提示符是不是 `pi@pi-zero:~ $`？如果是，说明是树莓派窗口。

**第一步：复制 .env.example 为 .env**

```
① 在树莓派窗口复制粘贴下面命令：
   cd ~/robot
② 按键盘回车键
   没有输出（直接出现提示符）—— 说明已经进入 robot 文件夹
③ 复制粘贴下面命令：
   cp .env.example .env
④ 按键盘回车键
   没有输出 —— 说明复制成功
```

> `cp` = copy（复制），把 .env.example 复制一份命名为 .env。

**第二步：用 nano 编辑器打开 .env 文件**

```
① 在树莓派窗口复制粘贴下面命令：
   nano .env
② 按键盘回车键
   窗口界面变了 —— 进入了 nano 编辑器
   （看到 GNU nano 的界面，里面有 BAIDU_APP_ID= 等文字）
```

**你会看到这样的界面：**

```
  GNU nano 5.4                     .env                               
  
  # 百度语音识别/合成
  BAIDU_APP_ID=你的APP_ID
  BAIDU_API_KEY=你的API_KEY
  BAIDU_SECRET_KEY=你的SECRET_KEY
  
  # DeepSeek AI 对话
  DEEPSEEK_KEY=你的DEEPSEEK_API_KEY
  ...
```

**第三步：填写 Key**

用键盘的 **方向键**（↑↓←→）把光标移到 `你的APP_ID` 那里。

把 `你的APP_ID` 删掉（用退格键 Backspace），然后粘贴或输入你从百度复制的 AppID。

同样，把 `你的API_KEY`、`你的SECRET_KEY`、`你的DEEPSEEK_API_KEY` 都替换成真的 Key。

改完应该像这样：

```
  BAIDU_APP_ID=12345678
  BAIDU_API_KEY=abcdefghijklmn
  BAIDU_SECRET_KEY=1234567890abcdef
  DEEPSEEK_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

> 每行**不要有空格**，等号两边不要空格。格式是 `KEY=值`。

**第四步：保存并退出**

```
① 按住 Ctrl 键不放 → 按 X 键（会提示：是否保存？）
② 按 Y 键（确认保存）
③ 按回车键（确认文件名 .env）
```

> 这样就回到树莓派窗口的提示符了。

---

## 6.6 验证 Key 是否填对了

> 在 **🟢 树莓派窗口** 操作。

```
① 在树莓派窗口复制粘贴下面整段命令（鼠标选中 → Ctrl+C → 右键粘贴）：
   cd ~/robot && python3 -c "
   from dotenv import load_dotenv
   import os
   load_dotenv()
   print('百度 AppID:', os.getenv('BAIDU_APP_ID') or '❌ 没配置')
   print('DeepSeek Key:', '配置了 ✅' if os.getenv('DEEPSEEK_KEY') else '❌ 没配置')
   "
② 按键盘回车键
   看到类似这样：
   百度 AppID: 12345678
   DeepSeek Key: 配置了 ✅
   —— 说明 Key 配置成功
```

**✅ 两个都有值 → API Key 配置完成。**

如果看到 `❌ 没配置`，说明 Key 没填写正确，回到 6.5 重新检查。
