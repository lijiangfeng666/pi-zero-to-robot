## 🔍 每样东西长什么样？怎么挑？

### 1️⃣ 树莓派 4B（核心大脑）

树莓派就是一台**巴掌大的电脑**。你需要用它来跑机器人的程序。

**实物长什么样：**
- 大小：一张**银行卡**那么大
- 厚度：大概**两个硬币**叠起来
- 特征：板子上能看到很多金属针脚、一个 Type-C 电源口、两个 USB 口

**购买要点：**
- **一定要买 4B 型号**。3B/3B+ 性能不够，Zero 接口不全
- 内存 2GB/4GB/8GB 都可以，**推荐 4GB**（￥250 左右）
- 通常买到的是一块**裸板**（就一个电路板），需要自己买上面的配件

**收到货检查：**
```
☐ 电路板上印着 "Raspberry Pi 4 Model B"
☐ 有一个 Type-C 口（电源）
☐ 有两个 USB 口
☐ 有一个 HDMI 口（接显示器用的，我们用不上）
☐ 板子背面有 MicroSD 卡槽
```

**常见买错的情况：**
```
❌ 「树莓派 3B+」 → 太老，跑不动
❌ 「树莓派 Zero」 → 接口少，没 USB 大孔
❌ 「树莓派 400」 → 这是键盘电脑，不适合
❌ 「香橙派」「友善派」 → 不是树莓派，教程不通用
```

---

### 2️⃣ MicroSD 卡（装系统用的"硬盘"）

就是**手机用的那种小存储卡**。

**实物长什么样：**
- 比指甲盖大一点
- 正面有品牌 Logo 和容量数字（64GB）
- 反面有金色金属触点

**购买要点：**
- 品牌选 **闪迪（SanDisk）** 或 **三星（Samsung）**
- 速度选 **Class 10**（卡上会印着一个"10"字）
- 容量选 **64GB**（32GB 太小，128GB 浪费）
- **MicroSD**（最小的那种），不是普通 SD 卡（大个儿的）

> 💡 **常见误解：** 卖卡送的**透明小塑料壳**不是用来插电脑的，卡要插到**读卡器**才能连电脑。那个塑料壳只是保护套。

---

### 3️⃣ USB 读卡器

用来把 SD 卡插到电脑 USB 口上。

**实物长什么样：**
- 像**一个小 U 盘**，一端是 USB 接口（扁平金属片）
- 另一端有个**小卡槽**，刚好插 MicroSD 卡

**要不要买？**
- **笔记本用户：** 现在很多笔记本自带 SD 卡槽，看看你电脑侧面有没有
- **台式机用户：** 一般都需要买
- **Mac 用户：** 新款 Mac 需要买 Type-C 口的读卡器

---

### 4️⃣ 电源（树莓派的"充电器"）

**实物长什么样：**
- 跟安卓手机充电器差不多
- 接口是 **Type-C**（现在的手机充电口那种）
- 插头那一端是两脚扁插头（国标）

**购买要点：**
```
要满足：5V（电压）× 3A（电流）= 15W
```

- 搜 **"树莓派 5V 3A 电源 Type-C"**
- 不要用手机充电头！很多手机充电头虽然也是 Type-C，但功率不够，会导致树莓派重启、死机
- 专业的树莓派电源上面会印着 5.1V/3A

---

### 5️⃣ USB 麦克风（机器人的"耳朵"）

**实物长什么样：**
- 最常见的是 USB 麦克风**摄像头二合一**——一个带夹子的摄像头，上面有个小孔（麦克风）
- 也可以是独立的 USB 桌面麦克风
- USB 接口插到树莓派的 USB 口

**购买要点：**
- **最重要：必须写"免驱"** 或 "即插即用"（Plug and Play）
- 如果卖家说"要装驱动"——❌ 不要买，树莓派装不了
- 搜 **"USB 麦克风 免驱 会议麦克风"**
- 便宜的 ￥30 就能用，不用买贵的

**检查收到的货：**
```
☐ USB 插头（扁平金属片）
☐ 没有多余的驱动光盘/下载链接
```

---

### 6️⃣ USB 音箱（机器人的"嘴巴"）

**实物长什么样：**
- 一个小方盒子，带 USB 线（供电）和音频线（传声音）
- [搜"Y57 小扩音器"] 这个型号特别适合小白——一根 USB 线搞定供电+声音
- 也有插电的桌面音箱

**购买要点：**
- **首选 Y57 小扩音器**（￥20-30），一根 USB 线插树莓派就能用
- 次选 USB 供电的桌面小音箱
- ❌ **不要买蓝牙音箱**！蓝牙连树莓派很麻烦，延迟高，经常断
