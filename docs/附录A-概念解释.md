# 附录A：概念解释（小白必看）

> 本文档中出现的各种术语，如果你看不懂，来这里查。

---

## 💻 终端 / 命令行 / 控制台

**是什么：** 一个靠**输入文字命令**来操作电脑的界面，不是用鼠标点图标。

**长什么样：**
- **Windows 终端（PowerShell）：** 深蓝色背景，顶部白色文字
- **Mac 终端：** 白色（或黑色）背景，显示你的电脑名
- **树莓派终端（SSH 连接后）：** 显示 `pi@pi-zero:~ $`

**你只需要做：** 把文档里的命令**复制 → 粘贴进去 → 按回车**。

**常见误解：**
- ❌ "这不是黑客用的吗？" → 所有程序员都用这个，很普通
- ❌ "我输错了会不会把电脑搞坏？" → 文档里给的命令都是安全的

---

## 🔑 SSH（远程连接）

**是什么：** 一种协议，让你从**自己的电脑**连接到**树莓派**，远程控制它。

**好比：** 你用微信视频通话远程教爸妈用手机——SSH 就是电脑版的"远程控制"，只不过只能用文字命令操作。

**为什么需要：** 树莓派没有屏幕，你必须用 SSH 连上去才能操作它。

**怎么看自己是不是在 SSH 连接中：**
看一眼终端提示符的最后：
- `pi@pi-zero:~ $` → 你在树莓派里面
- `C:\Users\你的名字>` 或 `你的电脑名:~ $` → 你在自己的电脑上

---

## 🔑 API Key（应用程序接口密钥）

**是什么：** 一串**密码**，用来告诉云服务"我是谁，请允许我使用你的功能"。

**好比：** 去健身房办卡，每次去都要刷卡——API Key 就是那张卡。

**本项目用到的 API Key：**

| 服务 | 用来做什么 | 免费吗 |
|------|-----------|--------|
| 百度语音 Key | 语音识别（听懂你）+ 语音合成（开口说话） | ✅ 免费额度够用 |
| DeepSeek Key | 理解指令 + AI 对话 | ✅ 注册送免费额度 |

**注意事项：**
- API Key 相当于密码，**不要分享给其他人**
- 如果 Key 失效或额度用完，功能就不能用了，需要去对应平台续费

## 🎙️ 语音识别（STT）/ 语音合成（TTS）

**STT（Speech to Text）：** 把你说的话转成文字。
> 你说的"土豆放一首大海" → 百度 STT → 变成文字 `"土豆放一首大海"`

**TTS（Text to Speech）：** 把文字转成语音读出来。
> 机器人回答的文字 `"好的，正在播放"` → 百度 TTS → 变成语音从音箱播出来

---

## ☁️ 云服务

**是什么：** 别人公司的大型电脑（服务器），通过网络给你提供功能。

**本项目用到的云服务：**
```
百度智能云 → 语音识别（听懂你）+ 语音合成（说话）
     ↑
树莓派（你家） → 录音 → 发到百度 → 百度识别 → 返回文字

DeepSeek → AI 理解意图（你说"放一首大海" → 它理解成"播放音乐"）
     ↑
树莓派 → 把文字指令发给 DeepSeek → 返回意图解析结果
```

**所以树莓派必须联网才能工作。**

---

## 🧠 DeepSeek / AI 意图解析

**是什么：** 用 AI 来理解你说的自然语言。

**为什么需要：** 传统程序只能理解固定的命令（比如严格说"播放音乐"才能放歌），但 AI 可以说"放首歌听听"也能理解。

**在本项目中的实际效果：**
```
你说 → "土豆放一首大海"
DeepSeek 理解 → {"intent":"play_music","params":{"keyword":"大海"}}
机器人执行 → 搜索"大海" → 播放
```

## 🌙 唤醒词（Wake Word）

**是什么：** 一个**特定的词**，用来唤醒机器人开始听指令。

**好比：** 你叫你家狗的名字，它就会看你。

**本项目默认唤醒词：** `土豆`

**工作原理：**
```
① 安静等待（不听任何声音）
② 突然检测到声音 → 开始录音
③ 语音识别 → 如果是"土豆..." → 进入指令模式
④ 如果不是"土豆"开头 → 忽略，继续等待
```

**你可以改成任意词**（在 `.env` 文件中修改 `WAKE_WORD`），比如 `你好`、`小爱`、`哈利波特`。

## 🔊 VAD（语音活动检测）

**是什么：** 程序判断"现在有没有人在说话"的技术。

**在本项目中的作用：**
```
VAD 持续检测麦克风音量
  → 突然音量变大 → 有人说话了 → 开始录音
  → 音量恢复安静 → 话说完了 → 停止录音
```

**参数 `VAD_THRESHOLD`：**
- 默认 2500
- **越小越灵敏**（300 = 很灵敏，呼吸声都能触发）
- **越大越不灵敏**（8000 = 只有大声喊才能触发）

## 🎵 网易云 API 服务器

**是什么：** 一个在树莓派上运行的程序，负责"去网易云音乐搜歌"。

**它做了什么事：**
```
你说"土豆放一首大海"
  → 机器人解析意图 → 要放歌
  → 机器人问网易云服务器："搜'大海'"
  → 网易云服务器去网易云音乐官网搜
  → 返回歌曲列表
  → 机器人拿到歌曲链接 → 播放
```

**为什么需要这个：** 因为网易云音乐没有给树莓派直接用的搜索功能，所以需要在树莓派上装一个"中间人"（API 服务器）来转接。

## ⚙️ systemd / 系统服务

**是什么：** Linux 系统自带的服务管理工具。用来让程序在后台运行、开机自启。

**好比：** 手机上的"允许后台运行"设置。

**我们用 systemd 管理两个服务：**

| 服务名 | 作用 | 查看状态命令 |
|--------|------|-------------|
| `robot.service` | 机器人主程序 | `systemctl --user status robot.service` |
| `netease-api.service` | 音乐搜索服务 | `systemctl --user status netease-api.service` |

**常用命令：**
```bash
systemctl --user start 服务名     # 启动
systemctl --user stop 服务名      # 停止
systemctl --user restart 服务名   # 重启
systemctl --user enable 服务名    # 启用开机自启
systemctl --user disable 服务名   # 禁止开机自启
```

## 📂 路径缩写 `~` 和 `.`

| 符号 | 意思 | 举例 |
|------|------|------|
| `~` | 当前用户的主目录 | 在树莓派里就是 `/home/pi/` |
| `.` | 当前目录 | 你当前所在的文件夹 |
| `..` | 上级目录 | 当前文件夹的上一级 |

## 📋 常用 Linux 命令速查

| 命令 | 意思 | 举例 |
|------|------|------|
| `cd 文件夹名` | 进入这个文件夹 | `cd ~/robot` 进入 robot 文件夹 |
| `ls` | 列出当前文件夹的文件 | 看都有什么文件 |
| `cp A B` | 复制 A 为 B | `cp a.txt b.txt` |
| `mv A B` | 把 A 移动/重命名为 B | 改文件名、移动文件 |
| `rm 文件名` | 删除文件 | ❌ 小心用，删了找不回 |
| `mkdir 文件夹名` | 创建文件夹 | `mkdir myfolder` |
| `nano 文件名` | 用 nano 编辑器打开文件 | 改配置文件 |
| `sudo 命令` | 用管理员权限执行 | 装软件、改系统设置 |
| `Ctrl+C` | 取消/退出当前程序 | 机器人运行时按它退出 |
| `↑`（方向键上） | 上一条命令 | 不用重新打字 |
| `Tab` | 自动补全 | 输入 `cd ~/ro` 按 Tab → 自动变成 `cd ~/robot` |
