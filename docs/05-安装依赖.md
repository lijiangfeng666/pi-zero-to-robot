# 第五步：安装依赖

> 给树莓派装上机器人需要的软件。
>
> 📍 **在哪操作：树莓派窗口**（提示符是 `pi@pi-zero:~ $`）
>
> 如果你不小心关掉了树莓派窗口，重新操作一遍第4步的 SSH 连接就行。

---

## 5.1 确认你在树莓派窗口

先看一眼终端最下面一行的提示符是不是：

```
pi@pi-zero:~ $
```

如果是 → 继续往下做。

如果不是（比如显示的是 `lijiang@mycomputer:~ $` 或 `C:\Users\你的名字>`）→ 你还在电脑窗口。重新做第4.3步（SSH 连接）。

---

## 5.2 更新系统

> 让树莓派的系统保持最新，就像给手机系统更新一样。

在树莓派窗口里输入以下命令，然后按回车：

```bash
sudo apt update && sudo apt upgrade -y
```

> 这条命令的意思是：检查有没有新版本的软件（`update`）→ 如果有就全部升级（`upgrade -y`）。
> `sudo` 表示"用管理员权限执行"。
> 跑大约 **2 分钟**。

**运行过程中：**
- 屏幕会快速滚动很多文字——这是正常的
- 如果看到黄色字的警告——不用管，正常的
- 如果停在某个地方不动了——等一等，可能在下载

**等它跑完，重新出现 `pi@pi-zero:~ $` 提示符，就完成了。**

---

## 5.3 安装系统软件包

> 一次性安装机器人需要的各种软件。

**复制下面这整段命令**（从 `sudo` 到 `curl`），粘贴到树莓派窗口，按回车：

> 在终端里粘贴的方法：
> - 右键点击窗口的空白处 → 自动粘贴
> - 或者按 `Ctrl+Shift+V`

```bash
sudo apt install -y python3-pip python3-pyaudio python3-numpy python3-requests python3-lxml mpg123 pipewire pipewire-pulse wireplumber portaudio19-dev nodejs git curl
```

**这行命令很长，一次性复制粘贴进去就好。**

> 如果看到 `Is this ok [y/N]` 这样的提示，按键盘的 `Y` 再按回车。

**等待它跑完**（约 2-3 分钟），重新出现 `pi@pi-zero:~ $` 就完成了。

---

## 5.4 安装 Python 包

> 安装百度语音识别和 .env 配置文件读取功能。

在树莓派窗口输入：

```bash
pip3 install baidu-aip python-dotenv
```

按回车，等待安装完成。

**如果你在国内，网络可能比较慢。** 如果上面那条命令超过 30 秒没反应，先按 `Ctrl+C` 取消它，然后换成国内镜像源再试：

```bash
pip3 install -i https://pypi.tuna.tsinghua.edu.cn/simple baidu-aip python-dotenv
```

**看到 `Successfully installed` 就是安装成功了。**

---

## 5.5 创建项目目录

> 在树莓派里创建一个文件夹，专门放机器人的代码。

```bash
mkdir -p ~/robot
```

> 这条命令的意思：在用户主目录（`~`，也就是 `/home/pi/`）下创建一个叫 `robot` 的文件夹。
> 如果已经有这个文件夹了，它也不会报错（`-p` 参数的作用）。

按回车执行，没有输出是正常的。

---

## ⚠️ 如果你用的是树莓派 Zero（不是 4B）

> 本文档主要针对树莓派 **4B**。如果你用的是 **Zero 2W** 或 **Zero W**，请注意以下差异：

| 差异点 | 4B | Zero 2W / Zero W |
|--------|:--:|:----------------:|
| 性能 | 足够 | Zero W **性能较弱**，建议用 Zero 2W |
| Node.js | 上面命令直接装 | Zero W 需要手动装 armv6 版 |
| 麦克风 | USB 口多，随便插 | 需要 **USB OTG 转接头**（Type-C 转 USB） |
| 音箱 | USB 口多 | 同样需要转接头 |

**如果你用的是 Zero W（初代）：**

上面的 `apt install nodejs` 可能安装失败，因为官方源没有 armv6 的版本。手动安装：

```bash
# 先装 curl
sudo apt install -y curl

# 手动下载 Node.js v18（支持 armv6）
curl -sL https://unofficial-builds.nodejs.org/download/release/v18.20.4/node-v18.20.4-linux-armv6l.tar.gz -o /tmp/node.tar.gz

# 解压到 /usr/local
sudo tar -xzf /tmp/node.tar.gz -C /usr/local --strip-components=1

# 验证
node --version
```

> 其他步骤跟 4B 完全一样。如果你是 Zero 2W，上面 5.3 的 apt install 就能直接装 Node.js，不用手动。

## ✅ 依赖安装完成

| 装了啥 | 用来干嘛 |
|--------|---------|
| python3-pip | 装 Python 软件包的工具 |
| baidu-aip | 让树莓派能调用百度语音识别和合成 |
| python-dotenv | 读取 .env 配置文件 |
| mpg123 | 播放 MP3 音乐 |
| pipewire/wireplumber | 音频系统（让声音正常输出） |
| nodejs/git/curl | 后面部署会用到的工具 |

**✅ 确认一下：最后一个命令执行完后，提示符回到了 `pi@pi-zero:~ $` → 依赖安装完成，进入下一步。**
