# 第三步：烧录系统

> 把操作系统装到 SD 卡上，就像给新电脑装 Windows。
>
> 📍 **在哪操作：** 全程在你的 **电脑**（Windows/Mac）上操作，还没用到树莓派。

---

## 3.1 下载烧录工具

**① 打开浏览器**

- **Windows 电脑：** 双击桌面的 Chrome 或 Edge 图标（蓝色或彩色的圆形图标）
- **Mac 电脑：** 点屏幕上方 Launchpad（火箭图标）→ 点 Safari 浏览器

**② 访问下载页面**

在浏览器顶部的地址栏里，输入下面这个地址，然后按键盘上的 **回车键**：

```
https://www.raspberrypi.com/software/
```

**③ 下载安装包**

页面加载出来后：

- **Windows 用户：** 点蓝色的 "Download for Windows" 按钮
- **Mac 用户：** 点蓝色的 "Download for macOS" 按钮

下载的文件通常会出现在浏览器左下角（或右下角），文件名类似 `raspberry-pi-imager.exe`（Windows）或 `Raspberry-Pi-Imager.dmg`（Mac）。

> ⏳ 下载大约需要 1-2 分钟，等它走完。

**④ 安装烧录工具**

- **Windows 用户：** 找到下载的文件，**双击**它 → 弹出安装窗口 → 一直点「下一步」或「Next」→ 最后点「完成」
- **Mac 用户：** 找到下载的文件，**双击**它 → 把出现的 Raspberry Pi Imager 图标拖到 Applications（应用程序）文件夹

> 跟装微信、QQ 一样，一路默认安装就行。

---

## 3.2 开始烧录

**① SD 卡插到读卡器 → 插电脑 USB**

```
把 MicroSD 卡（小的那个）插到读卡器
→ 读卡器插到电脑的 USB 口
```

**② 打开 Raspberry Pi Imager**

- **Windows：** 点开始菜单 → 输入 `imager` → 点出现的「Raspberry Pi Imager」
- **Mac：** 点 Launchpad → 点「Raspberry Pi Imager」

**③ 选择系统**

```
点「Choose OS」按钮
→ 滑到 Raspberry Pi OS (other)
→ 点 Raspberry Pi OS Lite (64-bit)
```

⚠️ **注意：** 选 **Lite 版**（没有桌面图标的精简版），不要选 Desktop（带桌面的完整版）。Lite 版跑机器人更稳定。

**④ 选择 SD 卡**

```
点「Choose Storage」按钮
→ 点你的 SD 卡（看容量辨别，比如 64GB 那个）
```

---

## 3.3 ⚠️ 最关键一步：齿轮设置

**⑤ 点右下角的齿轮图标（⚙️）**

会弹出一个设置窗口。**必须填以下四项：**

| 设置项 | 填什么 |
|--------|--------|
| **Set hostname** | 填 `pi-zero`（项目名，表示"从零开始"，不是说你的树莓派是 Zero 型号） |
| **Enable SSH** | 勾上 → 选「Use password authentication」→ **用户名 `pi`，密码填 `raspberry`（全小写）** |
| **Set WiFi** | 勾上 → 填你家 WiFi 的名字和密码 → 国家选 `CN`（中国） |
| **Set timezone** | 勾上 → 填 `Asia/Shanghai` |

> ⚠️ **密码一定要设成 `raspberry`**，后面每一步都默认用这个密码，不要自己改。

填完点右下角的「SAVE」保存。

**⑥ 开始烧录**

```
点「WRITE」按钮
→ 弹出警告 "All existing data will be erased"（卡上数据都会被清除）
→ 点「YES」
```

**⑦ 等待进度条走完**

烧录大概需要 **3-5 分钟**。进度条会从左走到右，走到头就完成了。

**⑧ 烧录完成**

```
看到绿色的 "Write Successful"（烧录成功）
→ 点「CONTINUE」
→ 关闭 Raspberry Pi Imager
→ 在电脑右下角（或 Finder 里）弹出 SD 卡
→ 把 SD 卡从读卡器拔下来
```

**✅ SD 卡烧录完成，可以进入下一步了。**

---

## 3.4 常见烧录错误

| 你看到的 | 原因 | 怎么办 |
|---------|------|--------|
| Imager 不认 SD 卡 | 卡之前可能存过东西 | 搜"SD Card Formatter"下载安装 → 格式化 SD 卡 → 重试 |
| 忘了设 WiFi | 树莓派连不上网 | SD 卡插回读卡器，插电脑，重新烧一遍（记得填 WiFi） |
| WiFi 密码填错了 | 连不上网 | 同上，重新烧 |
| 烧录到一半报错 | 读卡器接触不好 | 拔插一下 SD 卡，重试 |

> **烧录成功后：** 把 SD 卡插到树莓派背面（金属触点是朝上的），**先别插电源**。
