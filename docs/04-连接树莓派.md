# 第四步：连接树莓派

> 树莓派没有屏幕，你需要通过 **SSH** 从你的电脑连上去操作。
>
> 📍 **第一步（4.1-4.2）：** 在树莓派上插卡开机
> 📍 **第二步（4.3）：** 在 **电脑** 上操作

---

## 4.1 插卡 + 开机

```
① 把烧好系统的 SD 卡插到树莓派背面
   —— 金属触点（金色部分）朝上，轻轻推进去，听到"咔嗒"一声
② 插电源线（USB-C 口）
   插上电后，看树莓派上的灯：
   红灯亮 = 供电正常
   绿灯闪烁 = 正在读 SD 卡（正常）
③ 等 1 分钟让树莓派启动完成
```

**如果灯不对劲：**

| 现象 | 原因 | 解法 |
|------|------|------|
| 红灯不亮 | 电源问题 | 换个充电器，树莓派需要 **5V/3A** 的电源 |
| 绿灯不闪 | SD 卡没插好或烧录失败 | 重新插 SD 卡，或者回第三步重烧系统 |

---

## 4.2 找到树莓派的 IP 地址（门牌号）

> IP 地址就是树莓派在网络上的门牌号，你的电脑要靠这个地址找到它。

**方法 A：从路由器后台看（推荐，最稳）**

路由器就是你家装宽带时送的那个（或你自己买的），上面有天线，连着网线。

```
① 在电脑上打开浏览器（Chrome/Edge/Safari）
② 在地址栏输入路由器地址，按回车
   常见地址：192.168.1.1 或 192.168.0.1
   打不开的话，看路由器背面贴纸上的地址
③ 输入路由器的管理员账号和密码（也在背面贴纸上）
④ 找「已连接设备」或「DHCP 客户端列表」
⑤ 找名字叫 pi-zero 或 raspberrypi 的设备
⑥ 记下它旁边的 IP 地址，格式类似：192.168.1.xxx
```

**方法 B：手机 Fing App（没有路由器密码时用）**

```
① 拿出手机
② 去应用商店搜「Fing」并安装
③ 打开 Fing，确保手机连的是你家 WiFi
④ 点「扫描」按钮
⑤ 等几秒，扫描结果里找 pi-zero 或 raspberrypi
⑥ 记下旁边的 IP 地址
```

**✅ 记下 IP 地址（比如 192.168.1.100），下一步要用。**

> 如果你用的是方法 B（Fing），记住这个 IP。后面第6步还要用一次。

---

## 4.3 SSH 连接——用电脑控制树莓派

> 现在你需要在电脑上打开一个 **终端（也叫命令行）**，用它连到树莓派。
> 这个窗口后面叫 **"树莓派窗口"**。

### Windows 用户——打开终端

```
① 按住键盘上的 Win 键（左下角四个方块的图标）→ 再按 R 键
   屏幕左下角会弹出一个小窗口叫「运行」
② 在运行窗口输入：powershell
③ 点「确定」或按回车
   会弹出一个深蓝色背景的窗口 —— 这就是终端
```

### Mac 用户——打开终端

```
① 点屏幕右上角的放大镜图标（Spotlight 搜索）
② 输入：终端
③ 点出现的「终端」应用
   会弹出一个白色（或黑色）背景的窗口 —— 这就是终端
```

### 输入 SSH 命令

```
① 在终端窗口输入下面命令（把 192.168.1.100 换成你的树莓派 IP 地址）：
   ssh pi@192.168.1.100
② 按键盘回车键
   出现：pi@192.168.1.100's password:
   —— 系统在问你要密码
```

> `ssh` = Secure Shell（安全外壳），用来远程连接树莓派。

### 第一次连接——确认安全提示

**如果你是第一次连这个树莓派，输完命令按回车后，可能会看到：**

```
The authenticity of host '192.168.1.100 (192.168.1.100)' can't be established.
Are you sure you want to continue connecting (yes/no)?
```

```
① 在光标处输入：yes（全小写）
② 按键盘回车键
   出现：pi@192.168.1.100's password:
   —— 继续输入密码
```

> 这是正常的，只有第一次连接才会出现。意思是"你确定要连这台机器吗？"

### 输入密码

```
pi@192.168.1.100's password:
```

```
① 在光标处输入树莓派的密码：raspberry（全小写）
② 按键盘回车键
   看到 pi@pi-zero:~ $ —— 连接成功！
```

> ⚠️ 输入密码时 **屏幕上什么都不会显示**，光标也不会动——这是正常的，电脑在保护你的密码。正常输入完按回车就行。

### 连接成功的标志

你会看到：

```
pi@pi-zero:~ $
```

> 看到这个提示符（`pi@pi-zero:~ $`）说明你已经成功连上树莓派了。
> **这个窗口就是"树莓派窗口"**。后面所有需要在树莓派上执行的命令，都在这个窗口里输入。

---

## 4.4 常见连接错误

| 你看到的 | 原因 | 怎么办 |
|---------|------|--------|
| `Connection timed out`（连接超时） | 找不到树莓派 | IP 错了？树莓派没开机？回 4.2 重新找 IP |
| `Connection refused`（连接被拒绝） | SSH 没开 | SD 卡插回电脑重新烧录（记得勾选 Enable SSH） |
| `Permission denied`（密码错误） | 密码输错了 | 重新输入 `raspberry`（全小写，没有大写字母） |
| `Host key verification failed` | 重装过系统 | 在电脑终端执行 `ssh-keygen -R 192.168.1.100`（换成你的 IP）再试 |

---

## 4.5 关于两个窗口——非常重要

从现在开始，你会有**两个窗口**：

| 窗口 | 怎么来的 | 长什么样 | 用来做什么 |
|------|---------|---------|-----------|
| **电脑窗口** | 你电脑上的终端（黑色的那个） | 显示你电脑的用户名@电脑名 | 传文件、下载代码 |
| **树莓派窗口** | 用 SSH 连上的（绿色的 pi@pi-zero） | 显示 `pi@pi-zero:~ $` | 安装软件、运行机器人 |

> 后面每一步都会告诉你"在哪个窗口执行"，一定要看清楚再输入命令。
